# server

## Depedencies

* express
* nodemon
* dotenv
* mongoose
* cors
* express-validator
* bcrypt
* dayjs
* morgan
* jsonwebtoken

## Api Endpoint

* `/api/auth/login` Login User
* `/api/auth/register` Register User

## Project setup
```
npm install
```

## Start Mongodb
```
sudo systemctl start mongod
```

## Set Env

copy `.env.example` to `.env` and edit

## Run

```
npm start
```
